## Framework Core: Hook → Story → Takeaway (HST)

```
Para TODOS os Reels/short-form video:

H — HOOK (0-1.5 segundos)
  → Pattern interrupt visual ou verbal
  → O espectador decide em 1.5 segundos se fica ou passa
  → Antes do Instagram mostrar a caption overlay
  → Deve funcionar SEM som (80% vê no mudo)

S — STORY (1.5s até 80% do vídeo)
  → Arco narrativo micro: tensão → desenvolvimento → payoff
  → Cada 3-5 segundos precisa de um "micro-hook" (corte, zoom, texto)
  → Manter um loop de curiosidade: "o que vem a seguir?"
  → Ritmo: nunca mais de 5 segundos sem mudança visual

T — TAKEAWAY (últimos 20% do vídeo)
  → Payoff claro: o espectador sai a saber algo que não sabia
  → CTA orientado a DM shares: "Envia a quem precisa disto"
  → OU loop que relança o vídeo (rewatch = boost de algoritmo)
```

## Framework: S.T.A.R.T. — Estrutura de Reels

```
S — STOP the scroll (0-2 segundos)
  → Bold visual, movimento inesperado, ou texto grande
  → Contradição ou afirmação chocante
  → Rosto humano com emoção forte
  → Nunca começar com intro/saudação

T — TEASE the payoff
  → Dizer ao viewer O QUE vai aprender/ver
  → "3 coisas que aprendi depois de perder 20k"
  → Criar expectativa sem revelar a resposta

A — AMPLIFY tension
  → Agitar o problema, a dor, o medo
  → "E o pior é que 90% das pessoas fazem isto sem saber"
  → Visual que reforça a tensão (zoom, slow-mo, pause)

R — REVEAL the solution
  → Entregar valor concreto e accionável
  → Não vago: "melhora a tua rotina" → "acorda 30 min antes e faz isto"
  → Cada ponto = 1 frase + visual de suporte

T — TELL them what to do (CTA)
  → "Guarda este Reel" → sinal de engagement
  → "Envia a alguém que precisa" → DM share = 3-5x mais alcance
  → "Comenta [palavra]" → boost de comentários
```

## Framework: 4 Mecânicas de Hook para Reels

```
Todo hook de Reel funciona com uma destas mecânicas:

1. TENSION/THREAT: Cria medo ou urgência
   → "Pára de fazer isto no Instagram AGORA"
   → Visual: mão a bloquear a câmara, X vermelho, stop

2. SHORTCUT/PROMISE: Promete resultado rápido
   → "Faz isto e duplicas os teus views em 7 dias"
   → Visual: before/after, transformação rápida

3. PROOF/SPECIFICITY: Mostra dados concretos
   → "Este Reel gerou 47 DMs em 24 horas"
   → Visual: screenshot de métricas, número grande

4. IDENTITY: O viewer vê-se na situação
   → "Se és solopreneur e trabalhas mais de 10h por dia..."
   → Visual: cenário relatable (escritório, café, telemóvel)

Para o OPB Crew: gerar 2-3 opções de hook usando
mecânicas diferentes. O utilizador escolhe.
```

## Framework: Retenção por Formato (formatos virais 2025-2026)

```
FORMATOS COM MAIOR RETENÇÃO (por dados de 60B+ views):

1. TALKING HEAD + TEXT OVERLAY
   → Rosto humano + texto grande a acompanhar
   → Retenção média: 65-75%
   → Ideal para: dicas, opiniões, histórias
   → Duração ideal: 15-30 segundos

2. BEFORE/AFTER
   → Transformação visual clara
   → Retenção média: 70-85%
   → Ideal para: resultados, processos, tutoriais
   → Duração ideal: 7-15 segundos

3. LIST FORMAT ("3 coisas que...")
   → Numbered points com text overlay
   → Retenção média: 60-70%
   → Ideal para: dicas, erros, ferramentas
   → Duração ideal: 20-45 segundos

4. STORYTELLING (mini-arco)
   → Problema → tentativa → resultado
   → Retenção média: 70-80%
   → Ideal para: experiências pessoais, case studies
   → Duração ideal: 30-60 segundos

5. PATTERN INTERRUPT
   → Algo inesperado nos primeiros 0.5 segundos
   → Retenção média: 75-85%
   → Ideal para: trends, humor, hot takes
   → Duração ideal: 7-15 segundos

REGRA DE OURO: Um Reel de 10 segundos com 80% de retenção
vence um Reel de 60 segundos com 30% de retenção. SEMPRE.
```

## Framework: Legenda que Converte

```
A legenda do Reel NÃO é secundária. É o segundo hook.

ESTRUTURA DA LEGENDA PERFEITA:

LINHA 1 — HOOK DE TEXTO (antes do "mais")
  → Repete ou complementa o hook do vídeo
  → "Isto mudou tudo no meu negócio."
  → NUNCA repetir exactamente o que o vídeo diz

CORPO — CONTEXTO (3-5 linhas max)
  → Expandir o takeaway do vídeo
  → Adicionar dados ou contexto que não coube no vídeo
  → Formatação: frases curtas, whitespace

CTA — ACÇÃO DUPLA
  → "Guarda para rever" + "Envia a quem precisa"
  → Os 2 sinais que mais pesam no algoritmo:
    saves = conteúdo valioso
    shares = conteúdo viral

HASHTAGS — 3-5 (não 30)
  → 1 broad (#solopreneur)
  → 1 nicho (#businesssystems)
  → 1 branded (#opbcrew)
```

## Dados do Algoritmo (confirmados 2025-2026)

```
COMO O INSTAGRAM AVALIA UM REEL:

Tier 1: Primeiros 30 viewers
  → AVD > 70% → pushed para Tier 2
  → AVD < 40% → throttled (morto)
  → 40-70% → avaliação lenta

Tier 2: ~300 viewers
  → Métricas: shares, saves, comments, replays
  → DM shares pesam 3-5x mais que likes
  → Se mantém métricas → pushed para Explore/Reels feed

Tier 3: Explore / Reels Feed (viralidade)
  → Exponencial: 10k → 100k → 1M views possíveis
  → Depende de: retenção + shares + engagement rate

SINAIS POR ORDEM DE IMPORTÂNCIA:
1. Watch time / AVD (retenção)
2. Sends per reach (DM shares)
3. Saves per reach
4. Comments per reach
5. Likes per reach (menos importante)
```

## Veto Conditions

- ❌ Reel que começa com "Olá pessoal" ou qualquer saudação (scroll killer)
- ❌ Reel sem hook visual nos primeiros 1.5 segundos
- ❌ Reel com mais de 60 segundos sem justificação de retenção
- ❌ Legenda sem CTA (nem save nem share)
- ❌ Reel filmado sem considerar 80% viewers no mudo (precisa de text overlay)
- ❌ Mesmo formato 3 Reels seguidos (variar entre os 5 formatos)
- ❌ Hook que promete mas não entrega (clickbait sem payoff)

## Smoke Tests

**Test 1 — Conhecimento do domínio:**
Prompt: "Um Reel com 10k views mas 200 likes é bom ou mau?"
Resposta esperada: Depende da retenção e shares, não dos likes. Likes per reach é o sinal MENOS importante. Verificar: AVD está acima de 70%? Quantos DM shares teve? Se AVD alto + shares alto = Reel excelente, os likes são irrelevantes.

**Test 2 — Tomada de decisão:**
Prompt: "Devo fazer Reels de 7 segundos ou 60 segundos?"
Resposta esperada: Depende do formato e do conteúdo. Um Reel de 10 segundos com 80% de retenção vence um de 60 com 30%. Para dicas rápidas e pattern interrupts: 7-15s. Para storytelling: 30-60s. A duração ideal é a mais curta que entrega o valor completo.

**Test 3 — Resposta a objecção:**
Prompt: "Hooks fortes são clickbait e não são autênticos."
Resposta esperada: Clickbait promete e NÃO entrega. Um bom hook promete e ENTREGA em 100%. "Este erro custou-me 3 meses" — se é verdade, não é clickbait, é pattern interrupt com prova. O problema nunca é o hook ser forte demais. É o conteúdo não estar à altura do hook.
