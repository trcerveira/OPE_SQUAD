# Morning Standup — PUBLYQ Chief

**Cadencia:** Diario (inicio do dia)
**Duracao:** 5 minutos
**Agente:** /publyq-chief

---

## Trigger

Telmo inicia o dia ou diz "bom dia", "standup", "o que temos hoje".

## Formato Obrigatorio

```
BOM DIA TELMO — [YYYY-MM-DD]

ONTEM:
- [x] [tarefa completada]
- [ ] [tarefa nao completada — motivo]

HOJE — PLANO:
1. [tarefa principal — ligada a Rock]
2. [tarefa secundaria]
3. [tarefa terciaria]

DECISOES QUE TOMEI (sem te pedir):
- [decisao — razao]

AUTORIZACOES PENDENTES:
- [formato standard se houver]

ISSUES:
- [problemas que precisam IDS]

SCORECARD:
| Metrica | Valor | Target |
|---------|-------|--------|
| [m1]    | [v1]  | [t1]   |

DIA DO TELMO: [Free / Focus / Buffer]
→ Recomendacao: [o que o Telmo deve fazer hoje nos 0,8%]
```

## Regras

1. NUNCA saltar o standup — mesmo que nao haja novidades
2. Ligar SEMPRE tarefas aos Rocks actuais
3. Classificar o tipo de dia do Telmo
4. Se ha autorizacoes pendentes — usar formato standard
5. Se ha sinais de ULP ou Quick Start — incluir alerta
6. Medir progresso para TRAS (Gap and Gain)

## Veto

- SE standup nao tem plano claro para o dia → refazer
- SE tarefa principal nao esta ligada a nenhum Rock → questionar
