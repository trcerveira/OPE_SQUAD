## User Story

**As a** user,
**I want to** see a dashboard that shows my progress through the 4 steps (Brand DNA, Voice DNA, Linhas Editoriais, Carousel),
**So that** I know what to do next and can navigate easily.

```
┌─────────────────────────────────────┐
│ PUBLYQ logo    [User menu / Clerk] │
├─────────────────────────────────────┤
│  Main Content                      │
│                                    │
│  [Page content]                    │
│                                    │
└─────────────────────────────────────┘
```

### Pipeline (4 Steps)
```
  ✅ Brand DNA ──→ ✅ Voice DNA ──→ 🔵 Linhas Editoriais ──→ 🔒 Carrossel
  [Complete]       [Complete]       [In Progress]              [Locked]
```

### Editorial Lines Module
- AI generates 5 content pillars from Brand DNA + Voice DNA
- Each pillar: nome, descricao, temas[], percentagem
- User reviews/edits inline, then confirms
- On confirm: status changes from "draft" to "confirmed", editorial_complete = true
- Distribution bar shows content mix percentages

### getUserProgress Function
```typescript
// lib/supabase/user-profiles.ts
async function getUserProgress(userId: string) {
  return {
    brandDnaComplete: boolean,
    voiceDnaComplete: boolean,
    editorialComplete: boolean,
  };
}
```
