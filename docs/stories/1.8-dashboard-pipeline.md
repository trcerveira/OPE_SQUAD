# Story 1.8 — Dashboard com Pipeline Progress (3 Passos)

> Epic: PUBLYQ MVP
> Priority: P0 | Points: 3
> Dependencies: Story 1.5, 1.6, 1.7
> Assigned to: @dev

---

## User Story

**As a** user,
**I want to** see a dashboard that shows my progress through the 3 steps (Brand DNA, Voice DNA, Carousel),
**So that** I know what to do next and can navigate easily.

---

## Acceptance Criteria

- [ ] Dashboard page at `/dashboard` (default after login)
- [ ] Dashboard layout with sidebar navigation (Brand DNA, Voice DNA, Carousel)
- [ ] Visual pipeline progress bar showing 3 steps:
  1. Brand DNA (complete/incomplete)
  2. Voice DNA (complete/incomplete, locked if Brand DNA incomplete)
  3. Carousel (complete/incomplete, locked if Voice DNA incomplete)
- [ ] Progress fetched from Supabase (`getUserProgress()`)
- [ ] Locked steps show lock icon + "Complete [previous step] first"
- [ ] Completed steps show checkmark + green accent
- [ ] Current step highlighted with accent color
- [ ] Quick action cards/buttons to navigate to each step
- [ ] Welcome message with user's name (from Clerk)
- [ ] Responsive: works on mobile and desktop
- [ ] Dashboard layout wraps all dashboard pages (brand-dna, voice-dna, carousel)

## Technical Notes

### Dashboard Layout (`app/(dashboard)/layout.tsx`)
```
┌─────────────────────────────────────┐
│ PUBLYQ logo    [User menu / Clerk] │
├──────────┬──────────────────────────┤
│ Sidebar  │  Main Content            │
│          │                          │
│ ● Brand  │  [Page content]          │
│ ○ Voice  │                          │
│ 🔒 Carousel │                       │
│          │                          │
└──────────┴──────────────────────────┘
```

Mobile: sidebar becomes bottom tab bar or hamburger menu.

### Progress Pipeline Component (`components/dashboard/ProgressPipeline.tsx`)
```
  ✅ Brand DNA  ──→  🔵 Voice DNA  ──→  🔒 Carousel
  [Complete]         [In Progress]       [Locked]
```

### getUserProgress Function
```typescript
// lib/supabase/user-profiles.ts
async function getUserProgress(userId: string) {
  // Query user_profiles for brand_dna_complete, voice_dna_complete
  // Carousel is "unlocked" when both are true
  return {
    brand_dna_complete: boolean,
    voice_dna_complete: boolean,
    carousel_unlocked: boolean
  };
}
```
