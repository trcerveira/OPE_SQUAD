## Technical Notes

### Deployment Steps (Telmo does in Vercel Dashboard)
1. Create new Vercel project
2. Import GitHub repo
3. Set framework: Next.js
4. Add all environment variables
5. Deploy
6. Configure custom domain: publyq.ai

### DNS Configuration
publyq.ai already points to Vercel (confirmed in PRD).
Verify A record and CNAME are correct.

### Clerk Configuration
Update Clerk dashboard:
- Add `publyq.ai` to allowed origins
- Set redirect URLs for production

### GitHub Actions (optional but recommended)
Copy CI workflow from OPB Crew (`.github/workflows/ci.yml`):
```yaml
name: CI
on: [push]
jobs:
  ci:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
      - run: npm ci
      - run: npm run lint
      - run: npm run build
```
