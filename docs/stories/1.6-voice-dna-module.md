> Priority: P0 | Points: 3

## User Story

**As a** solopreneur/PME owner,
**I want to** complete a Voice DNA assessment that captures how I write and speak,
**So that** all content generated sounds like me, not like a robot.

---

## Acceptance Criteria

- [ ] Voice DNA page at `/voice-dna` (protected route)
- [ ] GATED: Only accessible after Brand DNA is complete (`brand_dna_complete = true`)
- [ ] 8-question guided form (adapted from OPB Crew VozDNA)
- [ ] Completion time under 10 minutes
- [ ] On submit: sends answers to Claude API for Voice DNA generation
- [ ] Claude returns structured Voice DNA profile (JSONB)
- [ ] Profile saved to `voice_profiles` table in Supabase
- [ ] `user_profiles.voice_dna_complete` set to `true` on completion
- [ ] User can view their Voice DNA after completing
- [ ] User can re-do Voice DNA (updates existing, increments version)
- [ ] Loading state while Claude processes
- [ ] Error handling (API failure, rate limit)
- [ ] Rate limited: 5/day per user
- [ ] Audit logged (voice_dna_generate)
- [ ] Zod validation on all inputs

### Voice DNA Questions (8 questions — same as OPB Crew)
1. Qual e o teu nicho?
2. Qual e a tua oferta principal?
3. Qual e a maior dor do teu cliente ideal?
4. Como descreves o teu tom de comunicacao?
5. O que te diferencia dos outros?
6. Que palavras/expressoes usas frequentemente?
7. Que palavras/expressoes NUNCA usarias?
8. Cola aqui um exemplo de texto que escreveste (post, email, mensagem)

### Claude System Prompt (`lib/prompts/voice-dna.ts`)
Adapt from OPB Crew's existing Voice DNA prompt.

Output format (JSON):
```json
{
  "arquetipo": "communication archetype",
  "tom": { "warmth": 7, "formality": 3, "humor": 5 },
  "vocabulario_preferido": ["word1", "word2"],
  "vocabulario_proibido": ["word1", "word2"],
  "frases_tipo": ["example phrase 1", "example phrase 2"],
  "regras_de_voz": ["rule 1", "rule 2"],
  "estilo_resumo": "one paragraph voice summary"
}
```
