# Story 1.5 — Brand DNA Module (API + Componente + Prompt)

> Epic: PUBLYQ MVP
> Priority: P0 | Points: 5
> Dependencies: Story 1.2, 1.3, 1.4
> Assigned to: @dev

---

## User Story

**As a** solopreneur/PME owner,
**I want to** complete a guided Brand DNA assessment,
**So that** the AI understands my brand identity and can create content that represents me authentically.

---

## Acceptance Criteria

- [ ] Brand DNA page at `/brand-dna` (protected route)
- [ ] Multi-step guided form with questions about: brand name, mission, values, target audience, positioning, personality, differentiator
- [ ] Completion time under 15 minutes
- [ ] On submit: sends answers to Claude API for Brand DNA generation
- [ ] Claude returns structured Brand DNA profile (JSONB)
- [ ] Profile saved to `brand_profiles` table in Supabase
- [ ] `user_profiles.brand_dna_complete` set to `true` on completion
- [ ] User can view their Brand DNA after completing
- [ ] User can re-do Brand DNA (updates existing, increments version)
- [ ] Loading state while Claude processes
- [ ] Error handling (API failure, rate limit)
- [ ] Rate limited: 5/day per user
- [ ] Audit logged (brand_dna_generate)
- [ ] Zod validation on all inputs

## Technical Notes

### This is a NEW module (does not exist in OPB Crew)

### Brand DNA Questions (suggested — refine with Telmo)
1. What is your brand name?
2. What is your brand's mission? (1-2 sentences)
3. What are your top 3-5 brand values?
4. Who is your ideal client? (describe in detail)
5. What makes you different from competitors?
6. How would you describe your brand's personality? (e.g., bold, warm, professional, playful)
7. What tone do you want in your content? (formal ↔ casual scale)

### Claude System Prompt (`lib/prompts/brand-dna.ts`)
```
You are a brand strategist. Based on the user's answers, create a comprehensive Brand DNA profile.

Output format (JSON):
{
  "brand_essence": "one-line brand summary",
  "mission_refined": "polished mission statement",
  "values": ["value1", "value2", ...],
  "target_audience_profile": "detailed ideal client description",
  "positioning_statement": "X for Y that Z",
  "brand_personality": { "tone": "...", "energy": "...", "formality": "..." },
  "differentiator": "unique selling proposition",
  "content_themes": ["theme1", "theme2", ...],
  "avoid": ["things to avoid in content"]
}
```

### API Route (`app/api/brand-dna/route.ts`)
- POST: Validate → Rate limit → Claude API → Save to brand_profiles → Update user_profiles → Audit log
- GET: Fetch brand_profiles for current user

### Component (`components/brand-dna/BrandDNAAssessment.tsx`)
- Multi-step form (one question per screen or grouped)
- Progress indicator
- Submit → loading → result display
- Brand colors applied (dark theme)
