> Priority: P0 | Points: 5

## User Story

**As a** solopreneur/PME owner,
**I want to** generate Instagram carousels using my Brand DNA and Voice DNA,
**So that** I have professional content ready to publish that sounds like me.

---

## Acceptance Criteria

- [ ] Carousel page at `/carousel` (protected route)
- [ ] GATED: Only accessible after BOTH Brand DNA AND Voice DNA are complete
- [ ] 4-step wizard:
  1. Topic input (user types theme/topic)
  2. AI generates slides (5-10 slides with headline, body, CTA)
  3. Image selection (Unsplash picker per slide)
  4. Colors + Export (brand palette applied, download as images)
- [ ] Claude uses Brand DNA + Voice DNA as context for generation
- [ ] Generated carousel saved to `generated_carousels` table
- [ ] Content uses Voice DNA (tone, vocabulary, style — NOT generic)
- [ ] Last slide has CTA (call to action)
- [ ] Brand colors applied automatically to slide previews
- [ ] Export as individual PNG images (html-to-image)
- [ ] Export as ZIP with all slides (JSZip + file-saver)
- [ ] Unsplash image search integration
- [ ] Rate limited: 20/day per user
- [ ] Audit logged (carousel_generate)
- [ ] Zod validation on topic input

### Key Difference from OPB Crew
The carousel prompt MUST inject Brand DNA + Voice DNA:

```typescript
const brandProfile = await getBrandProfile(userId);
const voiceProfile = await getVoiceProfile(userId);

const systemPrompt = buildCarouselPrompt({
  brandDNA: brandProfile.brand_dna_output,
  voiceDNA: voiceProfile.voice_dna,
  topic: input.topic,
  keywords: input.keywords
});
```

### Claude System Prompt (`lib/prompts/carousel.ts`)
Adapt from OPB Crew's `carousel-premium.ts` but:
- Inject Brand DNA (positioning, values, themes)
- Inject Voice DNA (tom, vocabulario, frases_tipo, regras)
- Output: array of slides `[{ headline, body, cta?, imagePrompt }]`

### Pipeline Gating
```typescript
const progress = await getUserProgress(userId);
if (!progress.brand_dna_complete || !progress.voice_dna_complete) {
  redirect('/dashboard'); // Must complete both first
}
```

### Export Stack
- `html-to-image` — Convert slide DOM to PNG
- `jszip` — Bundle all PNGs
- `file-saver` — Trigger download
- Load these lazily (dynamic import) to keep bundle small
